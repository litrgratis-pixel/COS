# Sesje

Jedna sesja = jeden plik `YYYY-MM-DD-NNN.md` zawierający krótki zapis przebiegu, pakiety dowodowe i closer. Surową transkrypcję przechowuj tylko po redakcji i świadomej klasyfikacji danych.

Minimalny plik sesji:

```markdown
# SES-YYYY-MM-DD-NNN

Cel:
Tryb:
Źródła i commit:
Najważniejsze tezy:
Pakiet weryfikacyjny:
Wynik:
Closer:
```
