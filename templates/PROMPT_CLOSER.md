# Prompt closera

```text
Zamień tę sesję na propozycję zmiany stanu, nie na streszczenie.

Wypełnij dokładnie:
DECYZJE
ODRZUCONE
PARKING (z warunkiem powrotu)
BACKLOG
WIEDZA/DOWODY (fakty vs hipotezy)
AKTYWNY STAN
INCYDENTY/NIEZGODNOŚCI
NASTĘPNY KROK
KONTROLA ZAMKNIĘCIA

Nie twórz decyzji, której użytkownik nie zatwierdził.
Jeśli nic nie zmieniło stanu, zapisz NO_STATE_CHANGE i dlaczego.
```
